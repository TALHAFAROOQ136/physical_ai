# Module 3: NVIDIA Isaac - The AI-Robot Brain

NVIDIA Isaac provides GPU-accelerated simulation, perception, and AI capabilities for robotics development.

## Learning Objectives

By the end of this module, you will be able to:

- Set up Isaac Sim for photorealistic simulation
- Create synthetic data for AI training
- Implement GPU-accelerated perception pipelines
- Train robots with reinforcement learning

## The Isaac Ecosystem

| Component | Purpose |
|-----------|---------|
| **Isaac Sim** | Photorealistic physics simulation |
| **Isaac ROS** | GPU-accelerated ROS 2 packages |
| **Isaac SDK** | Robot development framework |
| **Isaac Lab** | RL training environments |

1. **[Isaac Sim Setup](/docs/module-3-isaac/isaac-sim-setup)** - Installation and configuration
2. **[Perception Pipelines](/docs/module-3-isaac/perception-pipelines)** - GPU-accelerated vision

## Prerequisites

- Completed [Module 2: Simulation](/docs/module-2-simulation)
- NVIDIA RTX GPU (RTX 2070 or better)
- NVIDIA Driver 525.60.11 or newer
