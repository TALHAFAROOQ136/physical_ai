# Physical AI اور Humanoid Robotics میں خوش آمدید

ذہین روبوٹس بنانے کے اس جامع کورس میں خوش آمدید! یہ نصابی کتاب آپ کو جدید humanoid robotics اور physical AI systems میں استعمال ہونے والی ضروری ٹیکنالوجیز اور تکنیکوں کی رہنمائی کرے گی۔

## آپ کیا سیکھیں گے

اس کورس کے اختتام تک، آپ یہ کرنے کے قابل ہوں گے:

- **روبوٹ کنٹرول سسٹمز بنائیں** ROS 2 کا استعمال کرتے ہوئے، جو robotics کے لیے industry-standard middleware ہے
- **حقیقت پسندانہ simulations بنائیں** Gazebo اور NVIDIA Isaac Sim کا استعمال کرتے ہوئے محفوظ testing کے لیے
- **AI perception سسٹمز implement کریں** vision، sensing، اور ماحول کی سمجھ کے لیے
- **language models کو integrate کریں** آواز اور text کے ذریعے قدرتی human-robot interaction کے لیے

## کورس کا ڈھانچہ

یہ کورس چار اہم modules اور ایک capstone project میں تقسیم ہے:

| Module | موضوع | مدت |
|--------|-------|----------|
| 1 | ROS 2 - روبوٹک اعصابی نظام | 2 ہفتے |
| 2 | Simulation - ڈیجیٹل جڑواں | 2 ہفتے |
| 3 | NVIDIA Isaac - AI-Robot دماغ | 2 ہفتے |
| 4 | VLA - Vision-Language-Action | 2 ہفتے |
| Capstone | اپنا Humanoid بنائیں | 4 ہفتے |

## یہ کس کے لیے ہے؟

یہ کورس ان کے لیے ڈیزائن کیا گیا ہے:

- **سافٹ ویئر انجینئرز** جو robotics میں منتقل ہونا چاہتے ہیں
- **Robotics کے شوقین** جو ذہین نظام بنانا چاہتے ہیں
- **AI پریکٹیشنرز** جو embodied AI اور physical systems میں دلچسپی رکھتے ہیں
- **طلباء** جو computer science، electrical engineering، یا mechatronics پڑھ رہے ہیں

## شرائط

شروع کرنے سے پہلے، آپ کے پاس ہونا چاہیے:

- بنیادی Python programming کی مہارت
- Linux command line سے واقفیت
- بنیادی ریاضی کی سمجھ (linear algebra، calculus)
- Ubuntu 22.04 (تجویز کردہ) یا WSL2 کے ساتھ Windows والا کمپیوٹر

Physical AI میں اپنا سفر شروع کرنے کے لیے تیار ہیں؟ آئیے [شرائط کی جانچ](/docs/intro/prerequisites) سے شروع کریں!
